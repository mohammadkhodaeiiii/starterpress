یک قالب سبک، مدرن و Full Site Editing برای وردپرس با تمرکز بر کارایی، دسترس‌پذیری، معماری تمیز و توسعه‌پذیری.

== Description ==

StarterPress یک قالب بلوکی (Full Site Editing) آماده تولید است که برای توسعه‌دهندگانی
طراحی شده که نقطه شروعی تمیز، سریع و توسعه‌پذیر می‌خواهند.

ویژگی‌ها:

* ویرایش کامل سایت با theme.json و قالب‌های بلوکی.
* معماری شی‌گرا SOLID با PSR-4 autoloading.
* بدون jQuery — فقط JavaScript خالص با بهبود تدریجی.
* دسترس‌پذیری در اولویت (WCAG 2.2 AA): لینک پرش، فوکوس، ARIA، صفحه‌کلید، prefers-reduced-motion.
* تمرکز بر کارایی: بارگذاری شرطی دارایی‌ها، کمترین CSS/JS.
* سازگار با RTL و فارسی با فونت وزیرمتن.
* آماده WooCommerce با استایل‌دهی بومی.
* چهار تنوع استایل: پیش‌فرض، روشن، تاریک و مینیمال.
* ۱۲ الگوی بلوکی باکیفیت، واکنش‌گرا و دسترس‌پذیر.

== Installation ==

1. در پیشخوان وردپرس به نمایش > قالب‌ها > افزودن > بارگذاری قالب بروید.
2. فایل zip قالب StarterPress را انتخاب و نصب کنید.
3. قالب را فعال کنید.
4. در صورت نیاز نمایش > ویرایشگر را برای سفارشی‌سازی باز کنید.

== Frequently Asked Questions ==

= آیا StarterPress به افزونه نیاز دارد؟ =

خیر. به‌صورت پیش‌فرض کار می‌کند. WooCommerce اختیاری است.

= آیا قالب فارسی و RTL است؟ =

بله. تمام متن‌های پیش‌فرض فارسی و جهت RTL از ابتدا فعال است.
