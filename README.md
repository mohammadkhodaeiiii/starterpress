# StarterPress

A lightweight, modern Full Site Editing (FSE) WordPress theme built for developers with a focus on performance, accessibility, clean architecture and extensibility.

## Features

- Full Site Editing (FSE)
- theme.json support
- Block Patterns
- Block Styles
- WooCommerce ready
- RTL support
- Accessibility ready
- Translation ready
- No jQuery
- Vanilla JavaScript
- SOLID architecture
- PSR-4 autoloading
- Four style variations

## Requirements

- WordPress 6.8+
- PHP 8.0+
